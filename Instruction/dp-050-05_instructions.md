# DP-050 - SQL-Workloads から Azure への移行
# ラボ 5 - SQL Workloads を SQL Database Managed Instance に移行する

**開発中**

>**重要**：Azure SQL Database Managed Instance の作成には、数時間かかる場合があります。現在、Azure SQL Database Managed Instance への移行をよりタイムリーに実行する方法を確認できるラボを提供する方法に取り組んでいます。さて、次の [Microsoft Cloud Workshop](https://handsonlabs.microsoft.com/handsonlabs/SelfPacedLabs/?storyId=story://Content-Private/content/w-sqlmi-a/2migrate/migrate)で移行を実行する方法を探索します。 約 45 分の講義となります。ただし、トレーナーはこのラボ環境の管理を制御しません。

>**重要**：SQL マネージド インスタンスは、ビジネスおよびエンタープライズのお客様を対象とした大量の Azure リソースを消費するため、 個人サブスクリプションで作成することは **お勧めしません。** これを行う場合は、Azure の使用制限を監視する必要があります。
